Read4You Chrome Extension

An accessible text-to-speech Chrome extension for people with weak vision.

Features

  • Text-to-speech for selected text on any webpage
  • PDF reading support
  • Sentence-by-sentence navigation
  • High contrast mode
  • Playback controls (play, pause, next, previous, stop)
