# Read4You Test Data - GitHub Pages

This folder is published via GitHub Pages to host test data files for the Read4You Chrome extension E2E tests.

## Published URLs

- **HTML Sample**: https://aikeymouse.github.io/read4you_chrome/test-data/sample.html
- **PDF Sample**: https://aikeymouse.github.io/read4you_chrome/test-data/sample.pdf

## Setup

To enable GitHub Pages:

1. Go to repository Settings â†’ Pages
2. Select "Deploy from a branch"
3. Choose branch: `main`
4. Choose folder: `/docs`
5. Click Save

GitHub Pages will automatically publish the contents of this folder.

## Files

- `index.html` - Landing page with links to test data
- `test-data/sample.html` - HTML page for testing text selection and playback
- `test-data/sample.pdf` - PDF document for testing PDF reading functionality

## Note

The source test data files are maintained in `/tests/test-data/` and copied to `/docs/test-data/` for GitHub Pages hosting.
